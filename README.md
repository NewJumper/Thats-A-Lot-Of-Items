TALOI introduces a lot of decorative blocks, stairs, and slabs for building!
The mod is only available on Forge, there are no plans for Fabric, yet.

Changelog of TALOI can be found in releases, along with the mod file for the specific version. TALOI is available for download on [CurseForge](https://www.curseforge.com/minecraft/mc-mods/thats-a-lot-of-items).
